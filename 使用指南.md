# Claudia 使用指南

## 🎯 什么是 Claudia？

Claudia 是一个功能强大的 Claude Code GUI 应用程序，提供了一个直观的图形界面来管理和使用 Claude Code。它包含以下主要功能：

- **CC 项目管理**：管理您的 Claude Code 项目和会话
- **CC 代理创建**：创建和管理自定义 Claude 代理
- **使用情况追踪**：监控 API 使用情况和成本
- **MCP 服务器管理**：管理模型上下文协议服务器
- **时间线和检查点**：会话版本控制和历史管理

## 📋 主要功能使用指南

### 1. 项目管理 (CC Projects)

**访问路径**：`CC Projects → 选择项目 → 查看会话 → 恢复或开始新会话`

- **查看项目**：应用程序会自动检测您的 `~/.claude` 目录中的项目
- **管理会话**：每个会话都显示第一条消息和时间戳
- **恢复会话**：直接恢复之前的对话会话
- **开始新会话**：为项目创建新的对话会话

### 2. 代理创建 (CC Agents)

**访问路径**：`CC Agents → 创建代理 → 配置 → 执行`

**步骤**：
1. **设计代理**：设置名称、图标和系统提示
2. **配置模型**：选择可用的 Claude 模型
3. **设置沙盒配置文件**：应用安全限制
4. **执行任务**：在任何项目上运行您的代理

### 3. 使用情况追踪

**访问路径**：`菜单 → 使用情况仪表板 → 查看分析`

**功能**：
- 按模型、项目和日期监控成本
- 导出数据用于报告
- 设置使用情况警报（即将推出）

### 4. MCP 服务器管理

**访问路径**：`菜单 → MCP 管理器 → 添加服务器 → 配置`

**功能**：
- 手动添加服务器或通过 JSON 导入
- 从 Claude Desktop 配置导入
- 使用前测试连接

### 5. 时间线和检查点

**功能**：
- **会话版本控制**：在编程会话的任何时点创建检查点
- **可视化时间线**：通过分支时间线导航会话历史
- **即时恢复**：一键跳转到任何检查点
- **分叉会话**：从现有检查点创建新分支
- **差异查看器**：查看检查点之间的确切变化

### 6. CLAUDE.md 管理

**功能**：
- **内置编辑器**：直接在应用程序内编辑 CLAUDE.md 文件
- **实时预览**：实时查看 markdown 渲染效果
- **项目扫描器**：查找项目中的所有 CLAUDE.md 文件
- **语法高亮**：完整的 markdown 语法高亮支持

## ⚙️ 配置要求

### 必要条件
- **Claude Code CLI**：必须安装并在 PATH 中可用
- **Rust**：1.70.0 或更高版本
- **Bun**：最新版本
- **macOS**：11+ 版本

### 系统要求
- **内存**：最少 4GB（推荐 8GB）
- **存储空间**：至少 1GB 可用空间
- **操作系统**：macOS 11+、Windows 10/11 或 Linux (Ubuntu 20.04+)

## 🔧 故障排除

### 常见问题

1. **"claude command not found" 错误**
   - 确保 Claude Code CLI 已安装并在 PATH 中
   - 测试：`claude --version`

2. **编译错误**
   - 确保使用正确的编译器（clang 而不是 gcc）
   - 运行启动脚本会自动设置正确的编译器

3. **依赖问题**
   - 重新运行：`bun install`
   - 检查 Rust 安装：`cargo --version`

4. **性能问题**
   - 首次启动可能需要几分钟来编译 Rust 代码
   - 后续启动会更快

## 🛡️ 安全特性

Claudia 实现了多层安全保护：

1. **进程隔离**：代理在独立的沙盒进程中运行
2. **文件系统访问控制**：基于白名单的文件访问
3. **网络限制**：控制外部连接
4. **审计日志**：记录所有安全违规行为
5. **无数据收集**：所有数据都保存在本地机器上

## 💡 使用技巧

1. **首次使用**：启动后选择 "CC Projects" 或 "CC Agents"
2. **项目管理**：Claudia 会自动检测您的 `~/.claude` 目录
3. **代理创建**：从简单的任务开始，逐步创建更复杂的代理
4. **使用监控**：定期检查使用情况仪表板以控制成本
5. **备份重要会话**：使用检查点功能保存重要的对话状态
